📌 Overview

Tic-Tac-Win-Score is a modern and interactive Tic-Tac-Toe web application built with a premium UI approach. It features smooth animations, sound effects, and real-time score tracking, delivering an engaging and visually appealing gameplay experience.

✨ Key Features
🎨 Modern Glassmorphism UI – Clean dark-themed interface
⚡ Micro-Animations – Smooth transitions & hover effects
🏆 Score Tracking System – Tracks Player X vs Player O wins
🔊 Sound Effects – Audio feedback for moves & results
🎮 Interactive Gameplay – Real-time game state updates
📱 Fully Responsive – Works on mobile, tablet & desktop
🔄 Restart & Reset Options – Seamless replay experience
🛠️ Tech Stack
Layer	Technology
Frontend	React.js
Build Tool	Vite
Styling	Tailwind CSS
UI Library	shadcn/ui
Icons	Lucide React
