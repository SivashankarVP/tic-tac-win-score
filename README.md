# 🕹️ Tic-Tac-Win-Score

A premium, modern Tic-Tac-Toe application built with **React**, **Vite**, and **Tailwind CSS**.

Created by **Sivashankar**.

## ✨ Features

- **Modern UI**: Sleek, glassmorphism design with a dark aesthetic.
- **Micro-animations**: Smooth transitions and interactive hovered states.
- **Score Tracking**: Keep track of wins between Player X and Player O.
- **Sound Effects**: Immersive audio feedback for moves and game results.
- **Responsive**: Fully optimized for mobile, tablet, and desktop screens.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SivashankarVP/tic-tac-win-score.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd tic-tac-win-score
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

### Running the App

Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:8080`.
